# NoteApp

## Quick start

install dependency use pnpm

```
pnpm install
```

start project

```
npm run start
```

## What it implements?

-   [ ] Note Home
    -   [x] List Notes
    -   [ ] Search
        -   [ ] Search by title
        -   [ ] Search by content
        -   [ ] Search by tags
    - [x] Add Note
    - [x] Note detail
      - [x] Fetch data from indexdb
      - [ ] Delete Note
      - [x] Modify Note
